# ESILV Programs and Tracks
Source: https://www.esilv.fr/programmes

ESILV provides several engineering tracks focused on digital and emerging technologies.

## Core Areas
- Computer Science & Software Engineering
- Data Science & Artificial Intelligence
- Financial Engineering
- Mechanical Engineering & Modeling

## Notable Courses
- Algorithms and Data Structures
- Machine Learning
- Cloud Computing
- Numerical Methods
- Project Management

## Credits and Structure
- Each academic year totals 60 ECTS
- Program includes core modules, electives, projects, and internships
